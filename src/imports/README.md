# Architecture Portfolio Front-End

A modern, image-first architecture portfolio website for showcasing thesis/project boards, renders, floor plans, elevations, materials, and design concept.

## How to preview

Open `index.html` directly in a browser, or use VS Code Live Server.

## How to customize

1. Replace the placeholder contact details in `index.html`:
   - `architect@email.com`
   - `+63 000 000 0000`
2. Replace project/client name if needed:
   - `Nicco John R. Roma`
3. Add more projects by duplicating the `.feature-tile`, `.gallery-item`, or `.drawing-card` blocks.
4. Replace or add images inside the `assets` folder, then update the image paths in `index.html`.

## Included sections

- Fixed modern navigation
- Hero section with architectural renders
- Project highlights and metrics
- Featured work grid
- Hugpong design concept
- Image gallery with lightbox preview
- Technical drawings section
- Original presentation boards section
- Minimal contact section

## Build type

Static HTML + CSS + JavaScript. No backend required.
